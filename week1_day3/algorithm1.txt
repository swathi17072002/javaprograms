//algorithm to find the sum of first n even numbers

step1 : start
step2 : read the number n
step3 : initialize the  i<---2,sum<--0,count<--1
step4 : repeatedly execute the steps (i)-(iii) until(count<=n)
        i. sum<--sum+i
        ii. i<---i+2
        iii.count<--count+1
step5 : print sum
step6 : stop
